Power Bi

This project presents an interactive Power BI dashboard built using the Superstore dataset to analyze and visualize sales performance. It provides detailed insights into:

Total sales and yearly trends

Regional and category-wise performance

Top-selling products and low-performing periods

Business KPIs like total revenue, best region, and growth trends

The goal of this project is to transform raw data into actionable insights through effective data storytelling, helping stakeholders understand business growth patterns and make informed decisions.
